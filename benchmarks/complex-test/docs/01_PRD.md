# Product Requirements Document (PRD) - StressAgent Pro

## 1. プロジェクト概要
**StressAgent Pro**は、2028年のストレスチェック完全義務化（50人未満事業場）を見据えた、AI駆動型メンタルヘルスSaaSです。
従来の「年1回の調査」に加え、「日常会話AI」によるリアルタイムな予兆検知と改善提案を行います。

## 2. ターゲットユーザー
- **メイン**: 従業員10-49名規模の企業（IT, 広告, 介護, 運送）の人事/経営者。
- **課題**: 法改正への対応工数不足、離職率の増加、メンタル不調の早期発見遅れ。

## 3. コア機能 (MVPスコープ)
### A. ストレスチェック機能（法対応）
- 厚生労働省準拠の57項目アンケート実施機能（Web/SP）
- 未受検者への自動リマインド
- 集団分析レポートの自動生成（PDF）
- 高ストレス者の自動判定ロジック実装

### B. AIエージェント機能（差別化）
- **日常チャット**: Slack/Teams/LINE Works連携。Botから「調子はどう？」等のカジュアルな問いかけ。
- **会話分析**: 返答のテキスト、反応速度、絵文字から感情スコア（-1.0 ~ 1.0）を算出。
- **アラート**: ストレス値が閾値を超えた場合、管理者（または産業医）へ匿名性を担保しつつ通知。

### C. 管理者ダッシュボード
- 部署ごとのヒートマップ表示（ストレスが高い部署を可視化）
- AIによる改善アクション提案（例：「A部署は残業過多の傾向。ノー残業デーの導入を推奨」）

## 4. ユーザー体験フロー
1. **導入**: 管理者が従業員をCSV一括登録。
2. **日常**: 従業員はSlackでBotと数往復会話するだけ。
3. **定期**: 年1回、Web上で詳細な57項目チェックを受検。
4. **介入**: AIが高リスクを検知 → 管理者に「部署単位」でアラート → 対策実施。
