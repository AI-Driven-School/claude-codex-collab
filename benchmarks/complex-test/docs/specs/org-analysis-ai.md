## コンポーネント構成

| コンポーネント | 種類 | ファイル | 説明 |
|--------------|------|----------|------|
| OrgAnalysisPage | Page | `app/admin/org-analysis/page.tsx` | メインページ |
| OrgScoreCard | Component | `components/dashboard/OrgScoreCard.tsx` | スコアカード |
| AIInsightsPanel | Component | `components/dashboard/AIInsightsPanel.tsx` | AIインサイト表示 |
| DepartmentScoreList | Component | `components/dashboard/DepartmentScoreList.tsx` | 部署別スコア一覧 |
| TrendChart | Component | `components/dashboard/TrendChart.tsx` | トレンドグラフ |

---

## 状態遷移

| 状態 | トリガー | 遷移先 |
|------|---------|--------|
| 初期表示 | ページロード | ローディング中 |
| ローディング中 | API応答成功 | データ表示 |
| ローディング中 | API応答失敗 | エラー表示 |
| データ表示 | レポート出力ボタン | PDF生成中 |
| PDF生成中 | 生成完了 | ダウンロード開始 |

---

## インタラクション

### PDFレポート出力
1. ボタンクリック
2. ローディング表示
3. API呼び出し (`POST /api/v1/admin/org-analysis/generate-report`)
4. PDF URLを取得
5. 自動ダウンロード開始

### 部署クリック
1. 部署行をクリック
2. 部署詳細モーダル表示
3. 詳細API呼び出し
4. 部署固有のAI分析表示

---

## スタイリング

既存のデザインシステムに従う:
- カード: `card` クラス
- ボタン: `btn-primary`, `btn-secondary`
- カラー:
  - 高リスク: `text-red-500`
  - 中リスク: `text-yellow-500`
  - 低リスク: `text-green-500`
- アニメーション: `animate-fade-in`
